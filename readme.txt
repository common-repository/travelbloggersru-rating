== Description ==

This plugin adds a counter to your site that is needed for [Travelbloggers.ru rating](http://travelbloggers.ru/rating.html "Russian Travel Bloggers Rating").

== Installation ==

1. Go to your "Dashboard" -> "Plugins" -> "Add new"
2. Search for "Travelbloggers.ru Rating"
3. Press "Install"
